### Moderator 生成的 PRD 基础版

```
【Moderator】# PRD：AIChat网站Sandbox模式（基础版）

## 1. 目标用户
*   **开发者**：用于快速原型验证、自动化文档生成或测试AI代理的工具调用能力。
*   **学生**：用于辅助完成课程报告、演示文稿等学习任务，通过自然语言交互简化文件创建流程。

## 2. 核心功能
1.  **Sandbox环境**：提供一个安全的、隔离的代码执行环境，供AI Agent调用工具。
2.  **工具集成**：
    *   **PPT生成工具**：集成`python-pptx`库，支持创建幻灯片、添加文本/形状、插入图片、应用模板等。
    *   **Markdown笔记工具**：支持创建、编辑.md格式的文本文件。
3.  **自然语言交互**：用户可在AIChat网站中直接通过聊天界面，以自然语言描述需求。
4.  **文件输出与下载**：生成的`.pptx`或`.md`文件可供用户预览或下载。

## 3. 解决的问题
*   降低使用门槛：用户无需学习编程语法，通过自然语言对话即可生成结构化文档。
*   提升创作效率：将重复性的文档格式编辑工作自动化。
*   提供安全试验场：为开发者和学生提供安全的沙盒环境。

## 4. 成功指标（MVP阶段）
*   功能完成度：100%支持通过自然语言指令生成包含文本、图片和基础排版的`.pptx`文件。
*   任务成功率：用户发起指令后，系统能正确理解并输出可用文件的成功率 > 85%。
*   用户采纳：上线后首月，至少有20%的活跃用户使用过Sandbox模式功能。
*   安全运行：Sandbox环境运行期间，实现零敏感信息泄露事件。

## 5. 约束条件
*   安全约束：严格隔离，禁止网络出站访问（白名单除外）。
*   资源约束：执行时间<30秒，内存使用量<512MB。
*   范围约束（MVP）：仅支持生成PPTX和Markdown文件。
*   交互约束：基于现有AIChat网站的聊天界面，不单独开发图形化操作面板。

[ ✓ ] PRD基础版生成完成
[ ✓ ] 澄清阶段完成
```
